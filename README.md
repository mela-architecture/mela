# CK.MELA: Modular Executable Logical Architecture (V5.2 Open-Core)

### [WHY MELA?]
Эта система превращает обычный AI-чат в профессиональную рабочую среду, которая экономит до 10 часов в неделю и минимизирует риск логических ошибок и неверной интерпретации данных моделью (галюцинации)

**MELA (Модульная Исполняемая Логическая Архитектура)** — это открытая безкодовая система для управления контекстом и логикой AI, превращающая процесс взаимодействия с ней в структурированный рабочий поток.

---

## 🚀 Как запустить эту систему в чате с Gemini?

Чтобы активировать архитектуру, вам не нужны сервера или навыки программирования. Достаточно выполнить два шага:

### Шаг 1: Инициализация Ядра
Скачайте файл `ck.mela_core_v5.cfg` из этого репозитория и загрузите его в самом первом сообщении чата.
* **Что произойдет:** AI считает системные директивы, перестроит логику мышления, отсечет лишний шум и сфокусируется на выполнении ваших задач.

### Шаг 2: Подключение Профильного Модуля
Отправьте в этот же чат файл модуля (например, демонстрационный `ck_mod_demo.cfg`).
* **Что произойдет:** Ядро распознает модуль через Handshake-протокол, подгрузит его специфические инструкции и логику.

## 🛠 Структура репозитория
* `ck.mela_core_v7.cfg` — В разработке (WIP)

* `ck.mela_core_v5.cfg` — Стабильное ядро предыдущей версии (LTS).
* `ck_mod_demo.cfg` — Демо-модуль для ознакомления с архитектурой.
* ck_mod_skupka_v1.1.cfg — Специализированный модуль аудита и оценки техники.
* CK-MOD_DOCUMENT_AI_v6.txt — Скрипт/конфигурация для AI-обработки документов.
* CK-MOD_DOCUMENT_GENERATOR_v6.html — Веб-интерфейс для генерации документации в рамках архитектуры CK.MELA.
