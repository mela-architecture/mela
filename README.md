# CK_MELA: Modular Executable Logical Architecture (V5.2 Open-Core)

### [WHY MELA?]
Эта система превращает обычный AI-чат в профессиональную рабочую среду, которая экономит до 10 часов в неделю и минимизирует риск логических ошибок и неверной интерпретации данных моделью (галюцинации)

**MELA (Модульная Исполняемая Логическая Архитектура)** — это открытая безкодовая система для управления контекстом и логикой LLM, превращающая процесс взаимодействия с ней в структурированный рабочий поток.

---

## 🚀 Как запустить эту систему в чате с Gemini?

Чтобы активировать архитектуру, вам не нужны сервера или навыки программирования. Достаточно выполнить два шага:

### Шаг 1: Инициализация Ядра
Скачайте файл `mela_core_v5.cfg` из этого репозитория и загрузите его в самом первом сообщении чата.
* **Что произойдет:** ИИ считает системные директивы, перестроит логику мышления, отсечет лишний шум и сфокусируется на выполнении ваших задач.

### Шаг 2: Подключение Профильного Модуля
Отправьте в этот же чат файл модуля (например, демонстрационный `ck_mod_demo.cfg`).
* **Что произойдет:** Ядро распознает модуль через Handshake-протокол, подгрузит его специфические инструкции и логику.

## 🛠 Структура репозитория
* `mela_core_v6.cfg` — Системное «микроядро» стабильности. Отвечает за удержание контекста и выполнение команд.
* `mela_core_v5.cfg` — Системное «микроядро» стабильности. Отвечает за удержание контекста и выполнение команд.
* `ck_mod_demo.cfg` — Подключаемый демо-модуль. Интерактивная песочница, которая показывает работу архитектуры в действии.
* ck_mod_skupka_v1.1.cfg — Профессиональный модуль аудита и оценки техники на торговых маркетплэйсах.
